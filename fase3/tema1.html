Tema 1

Flex Box

Los Flexbox son módulos que permite o ayuda a distribuir el espacio entre los ítems de una interfaz. Se destaca el hecho de que el Flexbox maneja el layout en una sola dimensión a la vez ya sea como fila o como columna. Tiene la capacidad de adaptarse al entorno donde se visualiza

  • CSS Flexible Box Layout. (13 de Abril de 2022). En Wikipedia. https://es.wikipedia.org/wiki/CSS_Flexible_Box_Layout
