Propósito del sitio

Este sitio web tiene como propósito generar aprendizaje sobre el uso de los elementos HTML conocidos como Flexbox, Listas y Enlaces.

Temas que se publicaran

Los Flexbox son módulos que permite o ayuda a distribuir el espacio entre los ítems de una interfaz. Se destaca el hecho de que el Flexbox maneja el layout en una sola dimensión a la vez ya sea como fila o como columna.

Las Listas son un conjunto de elementos los cuales están precedidos por lo general por un número, estas pueden ser de tres tipos: Ordenadas, Desordenadas y de definiciones. En las listas ordenadas tenemos que se pueden ordenar por el tipo de datos que contienen y también de forma inversa. Las listas Desordenadas no toman en cuenta ningún tipo de orden para los elementos. Las listas de definiciones son en las que cada elemento toma una descripción o una definición, como si fuese una especie de diccionario

Los enlaces HTML permiten enlazar una página con otra para crearlo se debe hacer en el elemento A e indicar el destino con el atributo href, con el atributo title podemos colocar título al enlace
